# Accessories Research Prompt Template

Use this prompt with an LLM (like Perplexity, ChatGPT, or Claude) to research accessories for a new phone.

---

## Prompt for Cases Research

```
Research all available cases for the [PHONE MODEL] from major brands. For each case, provide:

1. Brand name
2. Case/product name
3. Features:
   - MagSafe/Qi2/Magnetic compatible? (Yes/No)
   - Port cover? (Yes/No)
   - Camera cover/slider? (Yes/No)
   - Kickstand? (Yes/No)
4. Direct purchase links (official brand site, Amazon, BestBuy if available)

Organize by protection level:
- **Hardened**: Rugged, military-grade, waterproof (Pelican, OtterBox Defender, UAG Monarch, i-Blason, SUPCASE, etc.)
- **Tough**: Good daily protection (Spigen Tough Armor, UAG Pathfinder, OtterBox Symmetry, Ringke Fusion X, etc.)
- **Basic**: Slim, minimalist (Spigen Thin Fit, official cases, PITAKA, Latercase, etc.)

Include brands: Pelican, OtterBox, UAG, Spigen, Ringke, Caseology, i-Blason, SUPCASE, Poetic, VRS Design, ZAGG, tech21, Mous, dbrand, QuadLock, GHOSTEK, ARMOR-X, Encased, CaseBorne, Rokform, Juggernaut, SaharaCase, TUDIA, bellroy, Peak Design, PITAKA, Thinborne, Latercase, Bare Cases, Google official cases.
```

---

## Prompt for Screen Protectors Research

```
Research all available screen protectors for the [PHONE MODEL]. For each protector, provide:

1. Brand name
2. Product name
3. Camera notch style: Cut Out, Covered, or Hole Punch
4. Material: Glass, Polymer, TPU, Ceramic Film, Hybrid
5. Warranty info if available
6. Direct purchase links

Organize by type:
- **Tempered Glass**: ZAGG Glass Elite, Spigen GlasTR, Whitestonedome Dome Glass, amFilm, TORRAS, etc.
- **Film/Polymer**: ZAGG Fusion, Spigen Neo Flex, Ringke Dual Easy Film, UAG Flex, PITAKA, etc.

Include brands: ZAGG, Spigen, Whitestonedome, Mous, Ringke, UAG, PITAKA, amFilm, TORRAS, ESR, Caseology, LK, Supershieldz, IQShield, ArmorSuit.
```

---

## Prompt for Chargers Research

```
Research compatible chargers for the [PHONE MODEL]. For each charger, provide:

1. Brand name
2. Product name
3. Wattage/Speed
4. Direct purchase links

Include:
- Official brand chargers
- Third-party fast chargers (Spigen, Anker, Belkin, UGREEN, Baseus)
- Wireless chargers (if supported)
- Car chargers
```

---

## Prompt for Miscellaneous Accessories

```
Research miscellaneous accessories for the [PHONE MODEL]:

1. **Camera Lens Protectors**: Spigen Optik, Ringke, Whitestonedome, ESR, ZAGG
2. **Skins**: dbrand, Slickwraps, EGO Tactical
3. **Mounts**: QuadLock, iOttie, Rokform, SP Connect (car, bike, desk)
4. **MagSafe/Magnetic accessories**: ESR HaloLock, Peak Design, Moment, Mophie

For each, provide brand, product name, and purchase links.
```

---

## Usage Notes

1. Replace `[PHONE MODEL]` with the actual device name (e.g., "Google Pixel 10 Pro XL")
2. Run each prompt separately for better results
3. Verify links are active before adding to guide
4. Check for MagSafe/Qi2 compatibility - important for magnetic case features
