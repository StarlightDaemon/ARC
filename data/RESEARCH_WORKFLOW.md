## Manual Workflow Steps

**For each device:**

1. **Open prompt file:**
   - Example: `data/research-prompts/Samsung/Galaxy-S25-Ultra-VERIFIED.txt`
   - Select all (Ctrl+A) and copy

2. **Run in Perplexity Pro:**
   - Go to perplexity.ai
   - Select **Kimi K2 Thinking** model from dropdown
   - Paste the entire prompt
   - Wait for response (30-60 seconds)

3. **Save results:**
   - Copy Kimi K2's entire response
   - Paste into the corresponding results file:
     - S25 Ultra → `data/researched/Samsung-Galaxy-S25-Ultra-Kimi-K2.md`
     - Z Fold 7 → `data/researched/Samsung-Galaxy-Z-Fold-7-Kimi-K2.md`

4. **Notify me:**
   - Let me know research is complete
   - I'll review the data quality
   - If good, I'll convert to JSON → BBCode + HTML
